Individual Website Project – Site Plan

Mining Engineering Career Guide

Site Name

Mining Engineering Career Guide

This name was selected because it clearly communicates the website’s purpose: to guide students and young professionals who are interested in pursuing a career in mining engineering.

Optional domain example: miningengineeringguide.org

Site Purpose

The purpose of this website is to provide clear, beginner-friendly information about mining engineering as a career path. The site will explain what mining engineers do, required education and skills, safety practices, technology used in mining, and career opportunities—particularly in Southern Africa and Zimbabwe.

Scenarios

  • What qualifications and skills do I need to become a mining engineer in Zimbabwe or Southern Africa?
  • What career opportunities are available for mining engineers locally and internationally?

Color Scheme

The following colors were selected to reflect professionalism, safety, and the industrial nature of mining.

  • Primary Color – Deep Blue (#1f3c88): Used for headings, navigation, and key branding elements.
  • Secondary Color – Gold/Yellow (#f4b41a): Used for accents, highlights, buttons, and emphasis.
  • Background Color – Light Gray (#f9f9f9): Used for page backgrounds to improve readability.

Typography

  • Montserrat (Sans-serif): Used for headings and navigation to create a strong, professional appearance.
  • Roboto (Sans-serif): Used for body text and paragraphs for readability and accessibility.

Wireframe – Home Page Layout

Mobile View (Small Screens)

  • Header with site name and hamburger navigation
  • Hero section introducing mining engineering
  • Overview section (What is Mining Engineering?)
  • Career highlights and opportunities
  • Call-to-action (Learn More / Explore Careers)
  • Footer with links and video demonstration

Desktop / Tablet View (Large Screens)

  • Header with horizontal navigation menu
  • Wide hero section with image and introduction text
  • Multi-column layout for education, skills, and careers
  • Featured technology and safety section
  • Footer with attributions and video link

Testing Plan

  • HTML Validation using W3C Markup Validator
  • Accessibility testing using WAVE
  • Performance, SEO, and Best Practices testing using Lighthouse
  • Color contrast testing using WebAIM Contrast Checker
